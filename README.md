# Admosphere New-Generation Site

This folder is the workspace for the React-based rebuild. The existing
`admosphere_1.html` remains unchanged as the visual reference.

## Component map

- `src/components/layout/` — floating navbar and footer
- `src/components/sections/` — hero, clients, services, process, work,
  testimonials, FAQ, contact, and final CTA
- `src/components/ui/` — buttons, cards, animated counters, backgrounds,
  badges, and other reusable primitives

## Planned page order

1. FloatingNavbar

3. ClientMarquee
4. StatsSection
5. ServicesBento
6. ProcessTimeline

Use cream surfaces, crimson accents, Syne headings, Roboto taglines, and
